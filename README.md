# B-Shell
A Classic **Shell created** for **basic purpose**, like **create folder/files**
# Setup
Firstly, you need to **install** **`_python_`** from [the Web site](https://www.python.org),
Secondly, execute 'Setup.bat' to upgrade python and install dependency,
Then, Open **main.py** with Python +1.X.X
And you can now use B-shell
# Help
Use the command help for get a idea of what each commands do.
