# Ketchup-Shell
The Best Open-Source Devs Shell for Windows.

# Directory Explained

### /KetchupShell.py
KetchupShell.py is the main file, He choose which command to run

### /Parser.py
Parser.py is the parser, it parse your command to make it readable for Python

## /Bin/
Bin contain all command, without /Bin/, the Shell is NOTHING.

# All commands explained
The shell use linux syntax for commands, You are maybe not family-friendly with linux command syntax
So here a review of every command

### Cat(Stands for Concatenate)
Show a file content
The file need to be in -file option

### Cd(Stands for Change Directoy)
Change the actual directory
New directory need to be in -path

### Clear
Clear the Shell

### Cp
Copy a file/directory from source(-src) to a destination(-dst)

### Disk
Create a .img Disk by a size of MB
Enter `disk -floppy` to create a Floppy Disk

### Echo
Print a text

### Grep
Show all line with a key-word in a fime

### Ls
Show all files and folders in current directory

### Mkdir
Make a directory

### Mv
Move a file from source(-src) to a destination(-dst)

### Pwd
Show the current directory

### Rm
Remove a File/Directory
Tips : Don't remove / or Windows lil vro >:3

### Runpy
Run a Python Script

### Specs
Show all your PC Specs

### Touch
Create a empty file

### Whoami
Show the current User
