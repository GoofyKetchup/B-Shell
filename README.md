# B-Shell

*A python* <b>Shell created </b>for **Devs use(Actually we are working on basic)**

# Setup(for older version)

Firstly, you need **to install *python*** from [The Web Site](https://www.python.org),

Secondly, execute 'Setup.bat' to upgrade python and install dependency,

Then, **Open main.py** with Python +1.X.X

And you can now use B-shell

# Help

Use the command help for get a idea of what each commands do.

If you break the shell by modifie it, use Reset.py in tools and reset folder.

# WARNING

The most older packages(v0.1, v0.2) is not compiled.
